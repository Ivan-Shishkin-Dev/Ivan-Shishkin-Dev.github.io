Ivan Shishkin Portfolio

A personal portfolio website built with HTML, CSS, and JavaScript. It showcases my experience, projects, and background as a Computer Science student at UC Irvine.

-------Overview-------
- Responsive layout
- Mobile-friendly navigation
- Background video layer
- Section-based design using HTML templates
- Lightweight JavaScript for content swapping and animations
- Styled with custom CSS (base, navigation, and section files)

-------Structure-------
- index.html
- script.js
- css/
  - base.css
  - nav.css
  - sections.css
- photo_video/
  - city_landscape.mp4
  - Headshot.JPG

-------Usage-------

Open index.html in any browser.
No build steps. No dependencies.

-------Deployment-------

Hosted through GitHub Pages.
Update the live site by pushing changes to the repository.

-------Purpose-------

Serves as my public, version-controlled personal website for employers and recruiters.
